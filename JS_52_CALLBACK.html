Chalo, isse detail mein samjhte hain: 1. Callback Function Kya Hai? Callback function wo function hai jo dusre function ke andar define kiya jata hai aur jab dusra function complete hota hai, tab ye call hota hai. Iska main purpose hai ki aap kisi operation ke complete hone par kuch specific code execute kar sakein.
2. Asynchronous Programming: JavaScript asynchronous programming ko support karta hai, jisme aapko kisi task ka result milne ka intezaar nahi karna padta. Ismein callbacks ka istemal hota hai taaki jab task complete ho jaye, tab aapko pata chale aur aap agla kaam kar sakein.
3. Example: Chalo ek simple example dekhte hain:
```javascript function greeting(name) { alert('Hello, ' + name); }
function processUserInput(callback) { var name = prompt('Please enter your name.'); callback(name); }
processUserInput(greeting);
```
. Function as Argument:
Jab aap kisi function ko dusre function ke argument ke roop mein pass karte ho, tab wo callback function hota hai. Jaise ki pehle wale example mein `greeting` function ko `processUserInput` function ke andar pass kiya gaya.
2. Function Call After Completion:
Callback function ko dusre function ke andar call kiya jata hai, usually kisi operation ke complete hone par. Agar aap dekhte ho ki ek function dusre function ke andar call ho raha hai aur uska result ya data use kar raha hai, to wo callback hai.
1. Function Definition:
`processUserInput` naam ka ek function hai jo ek parameter `callback` leta hai. Ye `callback` parameter ek function ko represent karta hai jo baad mein call kiya jayega.
2. User Input:
Is function ke andar, `prompt` function ka istemal kiya gaya hai jo user se naam puchta hai. Jab user apna naam enter karta hai, to wo `name` variable mein store hota hai.
3. Calling the Callback Function:
Uske baad, `callback(name)` likha gaya hai. Yahan, `callback` wo function hai jo aapne `processUserInput` function ko call karte waqt pass kiya hoga. Is line ka matlab hai ki jab user apna naam enter karega, to wo naam `callback` function ko pass kiya jayega.
Ab isko ek example se samjhte hain:
```javascript
function greet(name) {
console.log('Hello, ' + name + '!');
}

processUserInput(greet);
```
Yahan, `greet` function ko `processUserInput` function ke argument ke roop mein pass kiya gaya hai. Jab `processUserInput` function call hota hai:
- Pehle user se naam pucha jayega.
- Fir, jab user apna naam enter karega, to wo naam `greet` function ko pass kiya jayega.
- `greet` function phir `console.log` ke through greeting message print karega.


1. Define the Callback Function:
Sabse pehle, aapko ek function define karna hoga jo aap callback ke roop mein pass karenge. Ye function kisi bhi action ko perform kar sakta hai, jaise ki user input ko process karna ya koi message display karna.
